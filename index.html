Projetos em análise e ciência de dados

Desenvolvimento de um Painel Gerencial para Negócio com o Streamlit

Projeto Final: Fome Zero
Background de negócio: Marketplace - Food Delivery & Restaurants

Projeto de aplicação de conhecimento em Python com foco em análise de dados fazendo uso de uma base pública do Kaggle a informar as características dos estabelecimentos cadastrados de um marketplace global de restaurantes. O objetivo é mapear os estabelecimentos cadastrados, os tipos de serviços e culinárias oferecidos bem como analisar as avaliações alcançadas.O produto final escolhido foi um dashboard produzido e divulgado com o auxílio do framework Streamlit após os dados passarem por um processo de ETL.

Ferramentas utilizadas:

  • Jupyter Notebook e Jupyter Lab;
  • Tratamento e manipulação dos dados: Pandas e Numpy;
  • Visualização dos dados: Plotly, Matplotlib e Folium (visualização de dados georeferenciados);
  • Streamlit, Streamlit Cloud e GitHub.
